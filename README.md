# roboflow-labeler

Simple python script for auto-labeling using "Prompt with SAM3" roboflow feature.

## 🚀 Características
* **Automatización de Interfaz:** Control automático de mouse y teclado para agilizar tareas repetitivas de etiquetado.
* **Integración con SAM3:** Optimiza el proceso de anotación utilizando la función "Prompt with SAM3" de Roboflow.
* **Entorno Virtual Configurado:** Incluye una estructura limpia de entorno virtual (`pyvenv.cfg`) para aislar dependencias.

## 📁 Estructura del Proyecto
* `automatizacion_mouse_teclado.py`: Script principal encargado de ejecutar la lógica de automatización.
* `pyvenv.cfg`: Archivo de configuración del entorno virtual de Python.
* `LICENSE`: Licencia MIT de código abierto.

## 🛠️ Requisitos e Instalación
1. Asegúrate de tener Python instalado en tu sistema.
2. Clona este repositorio:
   ```bash
   git clone https://github.com
   cd roboflow-labeler
   ```
3. Activa tu entorno virtual e instala las librerías de automatización necesarias (como `pyautogui` o similares si las utilizas en el código):
   ```bash
   pip install pyautogui
   ```

## 💻 Uso
Ejecuta el script principal desde tu terminal asegurándote de tener abierta la ventana de la interfaz de Roboflow en tu navegador:
```bash
python automatizacion_mouse_teclado.py
```
